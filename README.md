# Student Performance Analysis

This project analyzes student habits and their impact on academic performance using data science techniques.

## Project Structure

- `app_main.py`: Main application script
- `app.py`: Additional application script  
- `Notebook.ipynb`: Jupyter notebook containing the data analysis
- `student_habits_performance.csv`: Dataset containing student habits and performance data

## Getting Started

1. Clone the repository
2. Install required dependencies (if any)
3. Run the analysis using the Jupyter notebook or Python scripts

## Dataset

The dataset `student_habits_performance.csv` contains information about student habits and their corresponding academic performance metrics.

## Analysis

The project explores correlations between various habits (such as study time, sleep patterns, extracurricular activities) and academic outcomes.
